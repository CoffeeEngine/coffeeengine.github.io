# Requirements

Before installing the project, make sure you have:

## System

- macOS, Linux, or Windows  
- Minimum 2GB RAM  
- Internet connection  

## Software

- **Hugo Extended** (latest version)
- Git
- Node.js (only if you use PostCSS or SCSS builds)

Check Hugo version:

```bash
hugo version
