# About This Project

This project is built using **Hugo** and the **Docsy** theme.  
It aims to provide clear, well-structured, and developer-friendly documentation.

## Our Goals

- Deliver high-quality documentation.
- Support multiple languages (English + Vietnamese).
- Make everything easy to navigate and search.
- Keep the site fast, lightweight, and mobile-friendly.

## Why We Build This

We want to provide a clean and modern documentation experience using:
- Hugo modules  
- Docsy features (search, dark mode, sidebar navigation)  
- GitHub Pages for simple deployment  

## Technologies We Use

- **Hugo** for static site generation  
- **Docsy** for layout, UI components, and docs structure  
- **GitHub Pages** for hosting  
- **Lunr.js** for search
